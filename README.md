## 📝 Descripción General
Sitio web desarrollado en HTML, CSS y JavaScript sin dependencias externas, diseñado para mostrar el catálogo completo de unidades Kenworth, promociones de tráilers, información de la empresa y formulario de contacto. Incluye sistema de búsqueda y lista de cotizaciones.

## 🖼️ Nomenclatura de Archivos e Imágenes
Todas las imágenes usan extensión **`.jpg`** para mejor compatibilidad y menor peso:

| Tipo de imagen          | Nombres exactos                          | Ubicación en el sitio       |
|-------------------------|------------------------------------------|-----------------------------|
| Identidad principal     | `logo.jpg`, `fondo-bienvenida.jpg`       | Todas las páginas / Inicio  |
| Modelos emblemáticos    | `emblematica1.jpg` a `emblematica4.jpg`  | Página de inicio            |
| Promociones             | `promocion1.jpg` a `promocion6.jpg`       | Página de promociones       |
| Catálogo completo       | `producto1.jpg` a `producto20.jpg`       | Página de productos         |

> ⚠️ **Importante**: Si cambias algún nombre de imagen, debes actualizar también la ruta dentro del archivo `.html` correspondiente para que no aparezcan espacios vacíos.

## 🛠️ Instrucciones de Instalación y Uso
### En tu computadora
1. Crea una carpeta nueva para el proyecto
2. Copia todos los archivos `.html`, `.css` y `.js` dentro de ella
3. Crea la carpeta `images` y guarda todas las fotos con los nombres indicados
4. Haz doble clic en `index.html` para abrir el sitio en tu navegador

### Para subirlo a internet
1. Ingresa a tu servicio de hospedaje web
2. Sube **todos los archivos y la carpeta `images`** tal cual están organizados
3. No muevas ni cambies la estructura de carpetas para evitar errores de carga

## 🎨 Personalización del Sitio
### Cambiar colores
- Abre `styles.css`
- Busca la sección correspondiente:
  - `.navegacion` → Barra de menú superior
  - `.pie-pagina` → Pie de página inferior
  - `.tarjeta-producto` → Tarjetas de unidades
  - `.btn-principal` → Botones principales

### Agregar nuevos modelos
1. Agrega la imagen con nombre consecutivo en la carpeta `images`
2. Copia el bloque de tarjeta en `productos.html`
3. Actualiza nombre, descripción y precio

### Modificar información
- Datos de contacto y dirección: en `contacto.html` y `sobre_nosotros.html`
- Precios y descripciones: dentro de cada archivo `.html` de productos
