# 🚛 Tráilers & Kenworth México
Sitio web oficial para venta, renta y financiamiento de unidades Kenworth y tráilers.

## 📋 Descripción
Sitio profesional desarrollado para mostrar el catálogo completo de unidades, promociones vigentes, información de la empresa y formulario de contacto. Incluye sistema de lista de cotizaciones y buscador de modelos.

## 📁 Estructura del proyecto

## ⚙️ Características
✅ Navegación sencilla y responsiva
✅ Buscador de modelos con resaltado de coincidencias
✅ Sistema de lista de cotizaciones
✅ Diseño moderno adaptado a celulares y computadoras
✅ Formulario de solicitud de información

## 🚀 Cómo ver el sitio
1. Sube todos los archivos a un repositorio de GitHub
2. Ve a **Configuración → Pages**
3. Activa GitHub Pages y en unos minutos tendrás tu sitio en línea
